# 🎮 Tic-Tac-Toe Game (C++ & SFML)

This is my very first solo game development project using C++ and the SFML library! Developing this Tic-Tac-Toe game was a challenging and exciting experience that helped me put my Object-Oriented Programming (OOP) knowledge into practice.

## 🚀 About the Project
I built this project to master the fundamentals of game loop implementation, input handling, and OOP concepts in C++. Being my first solo project, I am very proud of the effort I put into designing the game logic and structure from scratch.

## 🛠 OOP Concepts Applied
* **Encapsulation:** Managing the game state and board data securely within classes.
* **Inheritance:** Structuring game objects for better reusability.
* **Polymorphism:** Handling different game events efficiently.
* **Abstraction:** Keeping the main game loop clean and easy to understand.

## 💡 Why This Project Matters to Me
This project represents my first step into the world of game development and software engineering. I am deeply passionate about learning how to build interactive systems, and this project was the perfect playground to test my skills and problem-solving abilities.

## ⚙️ How to Play
1. Make sure you have [SFML](https://www.sfml-dev.org/) installed.
2. Clone this repository.
3. Compile and run the project using your preferred C++ IDE (I used Visual Studio).
